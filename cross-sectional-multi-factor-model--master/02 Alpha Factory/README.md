# 02 因子库 (Alpha Factory) 🏭

> 截面多因子量化选股系统 - 因子计算与数据清洗层

---

## ✨ 核心能力

| 能力 | 说明 |
|:---|:---|
| 📅 PIT对齐引擎 | 财务数据按公告日(m_anntime)对齐到交易日，避免未来函数 |
| 📈 TTM自动计算 | 流量表累计值→单季度→TTM滚动计算，处理跨年Q1特殊情况 |
| 🧹 四步清洗流程 | MAD去极值→缺失填补→OLS中性化→Z-Score标准化 |
| 📁 One Factor One File | 每个因子独立存储为Parquet宽表，已覆盖47个因子 |

## 🔄 数据流与逻辑

### 📅 PIT（Point-in-Time）对齐

**问题**：财务数据是季度报告，如何避免未来函数？

**方案**：按公告日（m_anntime）前向填充

```
财报1(2010Q1, 4/29公告) ─────── 财报2(2010Q2, 8/25公告)
     │                              │
     ▼                              ▼
2010-04-29 ~ 2010-08-24       2010-08-25 ~ 2010-10-27
   使用财报1数据                 使用财报2数据
```

**关键原则**：公告日后市场才知晓数据，因此公告日前只能用上一期数据。

### 📈 TTM计算逻辑

**问题**：利润表、现金表是累计值（半年报=Q1+Q2），如何转换为TTM？

**正确算法**：
```
步骤1: 累计值 → 单季度值
  - 跨年第一季度(3月): 单季度 = 累计值(新年Q1)
  - 同年内其他季度: 单季度 = 当前累计 - 上期累计

步骤2: 单季度 → TTM
  - TTM(t) = sum(单季度[t-3:t+1])  # 最近4个季度
```

### 🧹 因子清洗流程（四步顺序不能变）

```
原始因子（含极端值、行业偏误）
   ↓
1. MAD去极值（3倍MAD缩尾）      ← 必须先做
   ↓
2. 缺失值填补（行业中位数）      ← 缺行业/市值的剔除
   ↓
3. OLS中性化（剥离行业+市值Beta）
   ↓
4. Z-Score标准化（均值0，标准差1） ← 必须最后做
   ↓
清洗后因子（均值≈0，标准差≈1）
```

## 🔑 关键设计

### 技术因子 vs 财务因子

| 对比项 | 技术因子 | 财务因子 |
|:---|:---|:---|
| 数据来源 | 行情数据（日频） | 财务数据（季度） |
| 预处理 | 宽表转换 | TTM计算 + PIT对齐 |
| 计算逻辑 | 时序运算（shift/rolling） | 财务公式（乘除法） |
| 市值依赖 | 不需要 | 需要（close × cap_stk） |
| 未来函数风险 | 低（注意使用preClose） | 高（必须PIT对齐） |

### OLS中性化细节

**回归模型**：
```
因子 = β₀ + β₁×行业dummy₁ + ... + βₙ×行业dummyₙ + βₘ×对数市值 + ε
输出: ε (残差，即剥离行业+市值影响后的纯Alpha)
```

**共线性处理**：
- 小行业合并（少于2只股票的行业合并为"其他"）
- 移除常量行业列（避免完全共线）
- 变量数 >= 样本数时，只保留大行业

### 📁 One Factor One File

每个因子存储为独立的 Parquet 宽表：
- 行索引：日期 (datetime64[ns])
- 列：股票代码 (如 000001.SZ)
- 值：因子值 (float64，清洗后均值≈0，标准差≈1)

**优势**：
- 按需加载，减少内存占用
- 便于因子增删和版本管理
- 下游模型训练可灵活选择因子子集

## 📊 数据规范

### 因子宽表格式标准

| 维度 | 规范 |
|:---|:---|
| 行索引 | 日期 (datetime64[ns]) |
| 列 | 股票代码 (如 000001.SZ, 600000.SH) |
| 值 | 因子值 (float64) |
| 缺失值 | NaN |
| 文件格式 | Parquet |

### 技术因子字段（22个）

| 家族 | 因子 | 说明 |
|:---|:---|:---|
| 动量 | ret1/5/20/60/120 | N日收益率 |
| 动量 | ret20_60 | 动量差（ret20 - ret60） |
| 波动率 | std20/60 | N日标准差 |
| 波动率 | atr20 | 20日平均真实波幅 |
| 流动性 | amihud | Amihud非流动性 |
| 价量 | close_position | 收盘价日内位置 |

### 财务因子字段（25个）

| 家族 | 因子 | 说明 |
|:---|:---|:---|
| 估值 | pe/pb/ps/ey | 市盈率/市净率/市销率/盈利收益率 |
| 盈利 | roe/roa/opm/gross_margin | 净资产收益率/总资产收益率/营业利润率/毛利率 |
| 成长 | profit_growth/revenue_growth | 净利润/营收增长率 |
| 质量 | accrual | 应计利润比 |
| 安全 | debt_to_equity/current_ratio/cash_ratio | 产权比率/流动比率/现金比率 |
| 投资 | asset_growth/capex_to_assets | 总资产增长率/资本支出强度 |
| 效率 | asset_turnover | 资产周转率 |
